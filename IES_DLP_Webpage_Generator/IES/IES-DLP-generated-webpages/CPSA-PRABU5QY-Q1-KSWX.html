Going Deeper

This assignment was intended to help reinforce several concepts related to the training and evaluation of prediction models.

Beyond this, the code provided in this assignment is intended to provide examples of how to to build and apply these models in the future (so be sure to save a copy!)

If you are looking to go deeper with this code, aside from modifying other hyperparameters, there are additional models that you could also implement yourself that were not covered in this (but may be introduced in future modules) -- these are some common ones:

- K Nearest Neighbors
- Gradient Boosting Machines
- Stochastic Gradient Descent Classifier
- Random Forest
- Multilayer Perceptrons

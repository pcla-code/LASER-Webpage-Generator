Imagine you have a dataset of 10,000 learners taking a massive open online course (MOOC). This dataset records every page visit by these learners. Page types include the following: front_page, lecture, quiz, assignment, and forum. For each login session, there is a session id recorded in the dataset. Below is a sample of a few rows:

| timestamp | session_id | user | page_type |
| 2023-09-01 08:19:34 | 1 | user101 | front_page |
| 2023-09-01 08:19:39 | 1 | user101 | lecture |
| 2023-09-01 08:26:02 | 1 | user101 | quiz |

With the data, we hope to answer this question: Which pages did learners tend to visit together during a session? To answer this question, we will conduct mine association rules about page visits. In order to perform association rule mining, we need to get the most frequently occurring items in the dataset using the a priori algorithm.
